# Guild Leaderboards

The `GuildLeaderboards` component displays top guild rankings across multiple metrics sourced from the Albion Online Gameinfo API and cached through `guildService`.

## Usage

```tsx
import { GuildLeaderboards } from '@/components/pvp/guild-leaderboards';

export default function GuildDashboard() {
  return (
    <div className="grid gap-6">
      <GuildLeaderboards />
    </div>
  );
}
```
